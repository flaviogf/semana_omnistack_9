## :computer: Tecnologias

- [Node.js](https://nodejs.org/en/)

- [React Native](https://facebook.github.io/react-native/)

## :thinking: Como utilizar

### Backend

````sh
cd backend
````

````sh
yarn
````

````sh
yarn start
````

### Frontend

````sh
cd frontend
````

````sh
yarn
````

````sh
yarn start
````

### Mobile

````sh
cd mobile
````

````sh
yarn
````

````sh
yarn android
````
